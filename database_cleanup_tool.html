🔧 HRMS Database Cleanup Tool

⚠️ Important: This tool fixes database issues that can cause employee update errors. Run this before adding new employees or updating existing ones.

📊 Step 1: Check for Duplicate Employee IDs

Check if there are multiple employees with the same employeeId (this causes the update error you're seeing)

🔧 Step 2: Fix Duplicate Employee IDs

Automatically fix duplicate employeeIds by making them unique

🆔 Step 3: Generate Missing Employee IDs

Generate employeeIds for employees that don't have one

✅ Step 4: Sync Manager Data

Ensure all manager assignments are synced between Employee and User tables
