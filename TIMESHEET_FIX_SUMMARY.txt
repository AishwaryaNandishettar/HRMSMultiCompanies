🔍 ROOT CAUSE:
┌──────────────────────────────────────────────────────────────────────────┐
│ DATABASE STRUCTURE:                                                      │
│                                                                          │
│ ✅ Attendance Collection → Has empId field                               │
│    { empId: "GN-EMP-0007", name: "VishnuVardhan", ... }                  │
│                                                                          │
│ ✅ Employee Collection → Has employeeId field                            │
│    { employeeId: "IT-EMP-0041", fullName: "Lata Benakop", ... }          │
│                                                                          │
│ ❌ User Collection → Missing employeeId field                            │
│    { email: "lata.b@...", employeeId: null, ... }  ← PROBLEM!            │
│                                                                          │
│ BACKEND LOGIC:                                                           │
│ 1. Timesheet tries to get empId from Attendance → May be null           │
│ 2. Falls back to User.employeeId → Also null ❌                          │
│ 3. Generates hardcoded: "EMP-" + email_prefix ❌                         │
│    Result: "EMP-lata", "EMP-aishw", etc.                                │
└──────────────────────────────────────────────────────────────────────────┘

✅ SOLUTION:
┌──────────────────────────────────────────────────────────────────────────┐
│ SYNC User.employeeId FROM Employee.employeeId                           │
│                                                                          │
│ Method 1: Run MongoDB Script (Recommended - 5 minutes)                  │
│ ┌────────────────────────────────────────────────────────────────────┐ │
│ │ 1. Open MongoDB Shell:                                             │ │
│ │    mongosh "your_connection_string"                                │ │
│ │                                                                    │ │
│ │ 2. Run this script:                                                │ │
│ │    db.users.find().forEach(function(user) {                       │ │
│ │      var employee = db.employees.findOne({ email: user.email });  │ │
│ │      if (employee && employee.employeeId) {                       │ │
│ │        db.users.updateOne(                                        │ │
│ │          { _id: user._id },                                       │ │
│ │          { $set: { employeeId: employee.employeeId } }            │ │
│ │        );                                                          │ │
│ │      }                                                             │ │
│ │    });                                                             │ │
│ │                                                                    │ │
│ │ 3. Restart backend:                                                │ │
│ │    cd HRMS-Backend                                                 │ │
│ │    mvn spring-boot:run                                             │ │
│ │                                                                    │ │
│ │ 4. Test timesheet page                                             │ │
│ └────────────────────────────────────────────────────────────────────┘ │
│                                                                          │
