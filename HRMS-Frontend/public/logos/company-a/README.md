# Company A (TalentHub Solutions) Logo Assets

## Logo Requirements

### File Formats
- **Preferred**: SVG (scalable vector graphics)
- **Alternative**: PNG with transparent background

### Dimensions
- **Navigation Logo** (`logo.png`): 48x48 pixels or larger (will be scaled down)
- **Login Logo** (`logo-large.png`): 120x120 pixels or larger
- **Favicon** (`favicon.ico`): 16x16, 32x32, 48x48 pixels (multi-size ICO file)

### File Size Limits
- Navigation/Login logos: Maximum 500 KB
- Favicon: Maximum 100 KB

### File Names
- `logo.png` or `logo.svg` - Main navigation logo
- `logo-large.png` or `logo-large.svg` - Large logo for login screen
- `favicon.ico` - Browser favicon

Use these colors if creating logo assets:
- **Primary**: #1E40AF (Deep Blue)
- **Secondary**: #3B82F6 (Bright Blue)

- Logo should work well on both light and dark backgrounds
- Consider adding padding/whitespace around logo design
