# 🚀 Quick Start - Multi-Tenant HRMS

## Test in 3 Minutes!

### Step 1: Install Dependencies
```bash
cd HRMS-Frontend
npm install
```

### Step 2: Start Company A (Blue Theme)
```bash
npm run dev:company-a
```

### Step 3: Open Browser
```
http://localhost:5173
```

### Step 4: Verify
✅ Company name: "TalentHub Solutions"  
✅ Logo: "TH" in blue  
✅ Colors: Blue theme  
✅ Login works
